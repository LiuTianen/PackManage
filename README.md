# PackManage
多设备安装、卸载APK的脚本

环境说明：

需要先配置好Android SDK，python3.x的环境才能使用。

使用说明：

1、做了设备限制，区分手机和TV设备，对应的文件目录为：ApkInstall\Data。要将测试设备添加到设备列表里。

2、apk和aapt文件路径在 Config.ini里填写即可

3、点击start.bat即可运行程序
